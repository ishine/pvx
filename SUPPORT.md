# Support

## Getting Help

Use the following channels in order:

1. Read the README and docs under `docs/`.
2. Search existing issues and pull requests.
3. Open a new issue with reproduction details and exact command lines.

## What to Include in a Support Request

- Operating system and architecture
- Python version
- Exact command used
- Input/output file metadata
- Relevant console output and traceback
- Expected behavior vs observed behavior

## Scope

Support is best-effort for:

- Installation and setup
- CLI usage and parameter behavior
- DSP artifact diagnosis
- Documentation clarifications
